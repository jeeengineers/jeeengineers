================================================================================
                          PROJECT OVERVIEW
================================================================================

Jee Engineers is a professional business website for a leading manufacturer of
fly ash brick making machines and concrete block making machines based in 
Ahmedabad, Gujarat, India. The website showcases products, provides educational
content about fly ash bricks, and serves as a comprehensive resource for 
customers in the sustainable construction industry.

                        INSTALLATION & SETUP
================================================================================

1. REQUIREMENTS:
   - Web server (Apache/Nginx)
   - PHP 7.4 or higher (for contact form)
   - SSL Certificate (for HTTPS)

2. DEPLOYMENT:
   - Upload all files to web server root directory
   - Ensure proper file permissions
   - Configure .htaccess for Apache
   - Update contact form email settings in send.php
   - Test all forms and functionality

3. CONFIGURATION:
   - Update Google Analytics ID if needed
   - Configure email settings in send.php
   - Update social media links if changed
   - Verify sitemap.xml URLs
